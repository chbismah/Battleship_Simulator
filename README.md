# Battleship Simulator (C++)

A console-based **Battleship game** developed in **C++**, implementing core programming concepts such as arrays, functions, loops, conditionals, and basic game logic.  
The game supports **turn-based gameplay** with ship placement, hit/miss detection, and win conditions.
